# Музыкальный пак GoalCut

Положи сюда 4 MP3-файла без авторских прав:

- epic.mp3
- hiphop.mp3
- edm.mp3
- rock.mp3

## Где взять бесплатную музыку (без копирайта)

1. https://pixabay.com/music/  — поиск по жанру, скачивай MP3
2. https://freemusicarchive.org — фильтр CC0/Public Domain
3. https://www.bensound.com    — бесплатно с указанием автора

## Рекомендуемые треки с Pixabay

Ищи по тегам:
- epic → "epic cinematic sport"
- hiphop → "hip hop beats trap"
- edm → "electronic dance energy"
- rock → "rock guitar sport"

Скачивай файлы длиной 2-4 минуты (достаточно для любого клипа).
